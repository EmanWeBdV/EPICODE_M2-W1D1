## 📘 About the Project

This repository was created as part of the **Epicode Web Developer course**.

Unlike a practical coding project, this exercise is focused on **written answers** to basic theoretical questions related to programming and JavaScript fundamentals.

The goal of the exercise is to demonstrate understanding of beginner concepts through simple explanations written in your own words.

---

## 🧠 Topics Covered

The file includes answers about:

- **What an algorithm is**
- **What a variable is**
- **The difference between `undefined` and `null`**

These are fundamental concepts for anyone starting to learn programming and JavaScript.
